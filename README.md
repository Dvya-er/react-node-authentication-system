# 🔐 React + Node.js Authentication System

A full-stack authentication system built using React.js, Node.js, Express.js, MySQL, and JWT.

---

## Features

- User Registration
- Secure Login
- JWT Authentication
- Password Hashing (bcrypt)
- Protected Routes
- Logout
- Dashboard
- Role-Based Authentication

---

## Technologies

### Frontend

- React.js
- HTML5
- CSS3
- JavaScript

### Backend

- Node.js
- Express.js

### Database

- MySQL

### Authentication

- JWT
- bcrypt

## Installation

```bash
git clone https://github.com/username/react-node-authentication-system.git

cd frontend
npm install
npm start

cd backend
npm install
npm run dev
```
