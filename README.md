# ☁️ AWS EC2 Deployment Guide

A beginner-friendly step-by-step guide for deploying React.js and Node.js applications on AWS EC2 using Ubuntu, Nginx, PM2, and Git.

## 📌 Topics Covered

- Create AWS Account
- Launch EC2 Instance
- Configure Security Groups
- Connect via SSH
- Install Node.js
- Install Git
- Install Nginx
- Install PM2
- Clone GitHub Repository
- Deploy Backend
- Deploy React Frontend
- Configure Reverse Proxy
- Keep Application Running 24/7
